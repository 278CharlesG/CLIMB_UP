# 游戏核心愿景（Core Vision）
- **主题**：关爱留守老人，探讨现代年轻人在大城市事业追求与回乡陪伴父母之间的艰难抉择。
- **核心机制与心理学诡计**：在游戏中，玩家在城市工作、消费、晋升时，必须给予**极其强烈的正反馈**（华丽的特效、激昂的音效、实质性的金钱倍增、NPC的赞美、解锁炫酷道具）；而选择回乡陪伴父母时，只给予**微弱但温情的正反馈**（没有金钱奖励，只有柔和的色调、父母温馨的寒暄、简单的家常菜）。通过这种不平衡的诱惑，刻意引导玩家在游玩时“忽视”父母，从而在结局时产生强烈的心理冲击和自我反省。
- **美术风格**：参考《爱丽丝的摇篮》与《空洞骑士：丝之歌》，采用“梦幻类现实风格”。画面大量使用光影滤镜、漂浮的孢子、丁达尔效应，营造一种既真实又虚幻的氛围。

# 第二部分：游戏流程与剧情文本（请为以下每个阶段生成详细的对话剧本和UI文本）

## 【主角设定】
33岁，经历失业低谷后刚刚被大厂录取的青年。

## 【第一阶段：挣扎与初入大厂】（循环2-3次，每次跨度半年）
- **起点**：偏远农村父母家。
- **机制：繁琐的通勤**。通过一段不可跳过或需要频繁点击的动画（搭村镇公交 -> 转大巴 -> 转地铁），使用灰暗的色调，让玩家感到“回一趟家/去一趟城市真的很麻烦”。
- **机制：大厂打工小游戏**。模拟文件分类或敲击代码。完成后获得当天的工资。
- **机制：下班抉择**。
  - 选择【回家】：经历同样的繁琐通勤，到家后触发与父母的对话（请生成父母关心主角身体、舍不得吃穿的对话）。
  - 选择【城市消费】：进入城市商业街，可以用工资购买“高级西装”、“虚拟游戏机”等（提供绚丽的UI特效）。
- **时间推进**：每次循环时间跨度半年。

## 【第二阶段：城市立足与疏离】（循环2-3次，每次跨度1年）
- **触发事件**：主角逐渐适应，系统弹出提示：“通勤太累，是否在城市租房？（每月扣除一定租金）”。
- **机制变化**：如果租房，下班直接回出租屋，**免去繁琐通勤**。游戏节奏变快，更容易赚钱。
- **随机事件**（请生成文本）：
  - 父母打来电话慰问（玩家可选择“匆忙挂断”或“耐心听完”）。
  - 拿到高额奖金后，思考是否转账给父母（转账后收到父母“钱够用，你自己留着”的回复）。

## 【第三阶段：平步青云与迷失】（循环2-3次，每次跨度5年）
- **触发事件**：晋升管理层，在大城市买房。
- **机制：高压工作**。小游戏变得繁琐且限时，失败则扣发工资；但一旦成功，金钱奖励呈指数级增长，屏幕爆出大量金币特效和“优秀员工”奖状。
- **周末抉择**：
  - 【留在城市加班/社交】：获得地位提升、接触更高阶的消费场所（如私人游艇、高级酒会）。
  - 【回乡探望】：剧情表现出父母明显老去，行动迟缓（请生成3段不同的探望剧情，表现岁月的流逝）。

## 【第四阶段：结局分支】
- **主线事件**：主角正在主导一个关乎职业生涯的跨时代大项目，此时收到母亲短信：“你爸重病，速回。”
- **结局判断逻辑**：
  - **结局一（子欲养而亲不待）**：若前三个阶段玩家回乡次数极少。主角为了大项目留下。项目成功，获得极大的表彰动画。随后回家，场景变为【黑灰滤镜】的墓园。主角跪在坟前陷入死结。
  - **结局二（最后一面）**：若前三个阶段回乡次数一般（在事业与父母间摇摆）。面临终极二选一：选项目进入结局一；选回家进入医院场景（【冷色调滤镜】），在病床前陪伴父亲咽下最后一口气。
  - **结局三（防微杜渐的温暖）**：若前三个阶段频繁回乡。时间线强制提前半年！主角在探望时及时发现父亲的小病，强行带去体检。画面切入【暖黄色滤镜】，父亲病愈，一家人温馨吃饭。

## 【尾声：打破第四面墙】
- **神秘传送门**：无论哪个结局的最后，场景边缘散落着孢子光芒，出现一个梦幻的传送门。主角踏入后，屏幕渐渐变为纯白。
- **自白文本（请润色以下核心意思）**：“你以为这就是主角的故事吗？不……这是正在屏幕前的，你的未来。刚才经历的一切，是一次预演。现在，退出游戏，去给他们打个电话吧。”
